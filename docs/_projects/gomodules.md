# Go

The following are Go modules that are published for use in other applications.

## Utilities

- [gogitup](https://github.com/UnitVectorY-Labs/gogitup): Stop manually checking versions. Keep the tools you installed with go install updated with a single command.

## Modules

- [isplaintextfile](https://github.com/UnitVectorY-Labs/isplaintextfile): A lightweight Go module that determines whether a given file is plaintext by analyzing its content.
- [jsonschemaprofiles](https://github.com/UnitVectorY-Labs/jsonschemaprofiles): A Go library for validating and adapting JSON Schemas to constrained subsets used for predictable JSON output, such as LLM structured responses.
- [yamlequal](https://github.com/UnitVectorY-Labs/yamlequal): A lightweight Go library that verifies the semantic equality of YAML files.
